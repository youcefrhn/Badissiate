# دليل رفع تطبيق "بَادِيسِيَّات" على GitHub 🇩🇿

لتحويل هذا المشروع إلى تطبيق يعمل على الإنترنت، اتبع الخطوات التالية:

### 1. تجهيز المشروع على جهازك
- قم بتحميل المشروع (Export as ZIP) من واجهة AI Studio.
- فك الضغط عن الملف.

### 2. رفع المشروع إلى GitHub
- أنشئ مستودعاً جديداً (New Repository) على حسابك في GitHub.
- قم برفع جميع الملفات إلى المستودع (بما في ذلك مجلد `.github` الذي أنشأته لك).

### 3. تفعيل GitHub Pages
- اذهب إلى إعدادات المستودع (**Settings**).
- اختر **Pages** من القائمة الجانبية.
- تحت قسم **Build and deployment**، وفي خانة **Source**، اختر:
  - **GitHub Actions**.

### 4. النشر التلقائي
- بمجرد اختيار GitHub Actions، سيقوم الملف الذي أنشأته لك في `.github/workflows/deploy.yml` ببناء التطبيق تلقائياً.
- انتظر دقيقة، ثم اذهب إلى تبويب **Actions** في الأعلى لمتابعة عملية النشر.
- عند انتهاء العلامة الخضراء، ستجد رابط التطبيق جاهزاً!

### ملاحظة هامة بخصوص "الصفحة البيضاء":
إذا حاولت فتح ملف `index.html` من مجلد `dist` مباشرة على جهازك دون رفعه على خادم (Server)، فقد تظهر صفحة بيضاء بسبب قيود الأمان في المتصفحات (CORS). يجب دائماً تشغيله عبر خادم محلي مثل `live server` أو رفعه على GitHub كما هو موضح أعلاه.
